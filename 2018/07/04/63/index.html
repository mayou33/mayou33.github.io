系统管理员资源大全

2018-07-04 web web

kahun 在 Github发起系统管理员相关的开源资源整理。内容包括：备份/克隆软件、云计算/云存储、协作软件、配置管理、日志管理、监控、项目管理……

当然也有系统管理员相关书籍。

[https://github.com/kahun/awesome-sysadmin]

推荐一篇文章：《[10本适合于系统管理员的最佳书籍]》，目前比
kahun 他们整理的列表更丰富。

伯乐在线已在 GitHub 上发起「系统管理员资源大全中文版」的整理。
