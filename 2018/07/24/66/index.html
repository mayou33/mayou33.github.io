专业版：专业版是供商业性使用。它包括支持或附加功能，如无线并发连接等。

安装

rpm -ivh Nessus-7.1.2-es7.x86_64.rpm

启动

systemctl start nessusd.service

浏览器 https://IP:8834/

注意：Nessus服务使用的是https协议

在Linux系统中，Nessus工具默认安装在/opt/nessus目录中

自定义端口修改

/opt/nessus/etc/nessus/nessusd.conf.imported

获取激活码

选择免费的家庭版进行注册

https://www.tenable.com/products/nessus/activation-code

在使用Nessus之前，必须先激活该服务才可使用。如果要激活Nessus服务，则需要到官网获取一个激活码。下面将介绍获取激活码的方法。具体操作步骤如下所示：

在浏览器中输入以下地址：

http://www.nessus.org/products/nessus/nessus-plugins/obtain-an-activation-code

为了获取激活码。

在该界面First Name和Last Name文本框中，用户可以任意填写。

但是，Email下的文本框必须填写一个合法的邮件地址，用来获取邮件。

当以上信息设置完成后，单击Register按钮。

接下来，将会在注册的邮箱中收到一份关于Nessus的邮件。进入邮箱打开收到的邮件，将会看到一串数字，类似XXXX-XXXX-XXXX-XXXX，即激活码。

###################################

创建用户

输入激活码

要等一会（大约5分钟）

就完成了

更新

/opt/nessus/sbin/nessuscli update --all

systemctl restart nessusd.service

再登录页面，一会就能进入

扫描模板--------有upgrade的字样就是要收费的
