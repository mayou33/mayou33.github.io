yum4软件包管理器

2018-05-09 linux linux

基于DNF技术的YUM4，会正式的出现在即将发布的CentOS 7.5里。

CentOS 7.5系统里保留YUM4（/ usr / bin / yum4） 同时保留YUM版本3（/ usr / bin / yum）作为默认值。
他们都可以在同一个系统上使用，来回切换。

请注意，YUM4名称对于版本3和4的共存是暂时的

CentOS 8将会用YUM4作为默认值，并且使用（/ usr / bin / yum）作为默认值

三步安装，立即开始使用

yum -y install centos-release-yum4

yum -y install yum4

yum4 -y install dnf-plugins-core

cd /usr/bin 查看

yum4 -> dnf-2

dnf -> dnf-2

  1. yum -y install centos-release-yum4
  2. yum -y install yum4
  3. yum4 -y install dnf-plugins-core
