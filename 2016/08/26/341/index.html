<span style="font-family: times new roman,times,serif;"><span style="font-size: 16px;"><span style="color: rgb(51, 41, 10); background-color: rgb(255, 255, 255);"><span style="text-align: left; color: rgb(0, 0, 0); font-style: normal; font-weight: normal; background-color: rgb(255, 255, 255);">mainline =&gt; stable =&gt; longterm</span></span></span></span>

<span style="font-family: times new roman,times,serif;"><span style="font-size: 16px;">Mainline</span></span>



Mainline tree is maintained by Linus Torvalds. It's the tree where all new features are introduced and where all the exciting new development happens. New mainline kernels are released every 2-3 months.


<span style="font-family: times new roman,times,serif;"><span style="font-size: 16px;"><span style="color: rgb(51, 41, 10); font-style: normal; font-weight: bold; background-color: rgb(255, 255, 255);">Stable</span></span></span>


After each mainline kernel is released, it is considered "stable." Any bug fixes for a stable kernel are backported from the mainline tree and applied by a designated stable kernel maintainer. There are usually only a few bugfix kernel releases until next mainline kernel becomes available – unless it is designated a "longterm maintenance kernel." Stable kernel updates are released on as-needed basis, usually 2-3 a month.

<span style="font-family: times new roman,times,serif;"><span style="font-size: 16px;">Longterm</span></span>


There are usually several "longterm maintenance" kernel releases provided for the purposes of backporting bugfixes for older kernel trees. Only important bugfixes are applied to such kernels and they don't usually see very frequent releases, especially for older trees.
