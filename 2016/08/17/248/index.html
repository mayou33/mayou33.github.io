自定义docker CentOS 镜像

2016-08-17 docker docker

下载一个 CentOS 的镜像：



#&nbsp;docker&nbsp;pull&nbsp;centos:`7.2.1511`

下载完成后，使用命令查看本地镜像列表：

docker images centos


[root@localhost docker]#  docker images centos
