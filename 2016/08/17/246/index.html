docker centos7 安装ssh

2016-08-17 docker docker


一. 从docker hub 下载centos 官方镜像


docker pull centos:7

运行容器
docker run -i -t centos:7 /bin/bash
二. 安装passwd,openssl,openssh-server



yum -y install epel-release passwd openssh-server openssh-clients 


mkdir /var/run/sshd



ssh-keygen -q -t rsa -b 2048 -f /etc/ssh/ssh_host_rsa_key




ssh-keygen -t dsa -f /etc/ssh/ssh_host_ed25519_key
