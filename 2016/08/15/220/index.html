Docker修改镜像存放目录

2016-08-15 docker docker


docker的镜像默认存放位置是 / var / lib / docker 下，

 我们要在docker刚安装完成后，就修改成/data/docker,

方案一


创建目标目录

mkdir /data/docker

停止docker 服务

service docker stop

 vim /etc/sysconfig/docker


添加


OPTIONS='–graph=/data/docker –selinux-enabled –log-driver=journald'


然后在同步数据
mv -f /var/lib/docker/* /data/docker/

最后 service docker start

 

方案二


创建目标目录

mkdir /data/docker

停止docker 服务

service docker stop


mv -f /var/lib/docker/*  /data/docker/



rm -rf  /var/lib/docker


ln -s /data/docker  /var/lib/docker

最后 service docker start
