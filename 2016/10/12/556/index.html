Linux上查看硬件信息的一款神器 ：Inxi

2016-10-12 linux linux


源码：http://code.google.com/p/inxi/source/browse/trunk/inxi

moved to github


https://github.com/smxi/inxi


 

在线安装-centos系统

先安装EPEL：yum install epel-release -y

在安装inxi：yum install inxi

 

默认路径:/usr/bin/inxi

 

在线源码安装

要安装 inxi，只需执行下列命令即可：

git clone https://github.com/smxi/inxi.git

 

  chmod +x inxi

 

执行 ./inxi -c3 -v7 后，其输出结果如下，包括 CPU、图形、音频、网络、磁盘、分区等各方面的信息：

 

使用c选项，后面再跟上一个0到32之间的随机数。Inxi的输出结果标以颜色；

其中3是红色

 

输出到文本./inix -c3 -v7 > /tmp/sysinfo.txt
