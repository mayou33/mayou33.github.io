yum安装Percona Server on CentOS

2016-10-11 mysql mysql

系统要求：

#停止firewall

&nbsp;#禁止firewall开机启动

systemctl stop firewalld.service

systemctl disable firewalld.service

systemctl status firewalld

&nbsp;关闭selinux

setenforce 0

echo &quot;setenforce 0&quot; &gt;&gt; /etc/rc.local

sed -i &#39;s/SELINUX=enforcing/SELINUX=disabled/g&#39;&nbsp;&nbsp;&nbsp;&nbsp; /etc/sysconfig/selinux

echo never &gt; /sys/kernel/mm/redhat_transparent_hugepage/defrag

echo never &gt; /sys/kernel/mm/redhat_transparent_hugepage/enabled

echo never &gt; /sys/kernel/mm/transparent_hugepage/enabled

echo never &gt; /sys/kernel/mm/transparent_hugepage/defrag

1\. Install the Percona repository

yum -y install [https://www.percona.com/redir/downloads/percona-release/redhat/latest/percona-release-0.1-4.noarch.rpm](https://www.percona.com/redir/downloads/percona-release/redhat/latest/percona-release-0.1-4.noarch.rpm)

可以查看一下

yum list | grep percona

2\. Install the packages

&nbsp;

yum -y install Percona-Server-server-57
