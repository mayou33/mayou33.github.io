svn服务器安装

2016-11-30 devops svn


一 svn服务器

 

windows选择 VisualSVN Server -十分推荐的

 

linux选择 subversion

####

yum安装 （简单方便，版本低）



$ yum install subversion



 yum install mod_dav_svn 安装apache对svn的支持模块



yum install subversion-python  安装python对svn的支持


查看版本

svnserve –version
