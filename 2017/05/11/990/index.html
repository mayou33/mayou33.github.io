如何快速精确的和leader沟通

2017-05-11 粗制滥造 管理

2017-04-29  __58沈剑

<span style="font-size: 16px; background-color: inherit;">**【员工角度的潜在困惑，以及我作为leader内心的真实想法】**</span>

<span style="font-size: 14px; background-color: inherit;"><span style="color: rgb(244, 114, 7); font-size: 14px; background-color: inherit;">疑问一：leader时间很宝贵，会不会占用太多时间，导致影响leader工作节奏？</span>

**内心独白**：

事情确实不少，但**我存在的价值不就是帮助大家解决问题么**，期望你能尽快将问题描述清楚，我会综合手头要处理事情的重要紧急程度，判断是&ldquo;现场沟通&rdquo;，还是&ldquo;一会找你&rdquo;，或者&ldquo;某个时间点找我&rdquo;。</span>

&nbsp;

<span style="color: rgb(244, 114, 7); font-size: 14px; background-color: inherit;">疑问二：leader所说的&ldquo;一会找你&rdquo;是敷衍么？</span>

<span style="font-size: 14px; background-color: inherit;">**内心独白**：

兄弟，你想多了，&ldquo;一会找你&rdquo;就是字面的意思。</span>

&nbsp;

<span style="font-size: 14px; background-color: inherit;">一般来说，当手头有更重要的事情时，我会说&ldquo;一会找你&rdquo;。如无意外，这个事情将被写进schedule，稍后插空完成。极端情况下，如果没有找你，那是我真的忘了，请在即时通讯工具上，友善小窗提醒我，我一定会响应，并表达歉意。</span>

&nbsp;

<span style="color: rgb(244, 114, 7); font-size: 14px; background-color: inherit;">疑问三：leader为何会说&ldquo;某个时间点来找我&rdquo;？</span>

<span style="font-size: 14px; background-color: inherit;">**内心独白**：

一般比较正式，需要20分钟以上的沟通和讨论，会单独预留固定的时间和你一起商量把事情解决。</span>

&nbsp;

<span style="color: rgb(244, 114, 7); font-size: 14px; background-color: inherit;">千万不要迟到</span><span style="font-size: 14px; background-color: inherit;">，迟到可能打乱我的工作节奏。</span>

&nbsp;

<span style="font-size: 14px; background-color: inherit;">其他困惑？欢迎讨论。</span>

&nbsp;

<span style="font-size: 16px; background-color: inherit;">**【职场不会有人和你说】**</span>

<span style="color: rgb(244, 114, 7); font-size: 14px; background-color: inherit;">如何和leader快速+精准的沟通一件事？</span>

&nbsp;

**<span style="font-size: 14px; background-color: inherit;">一，表明期望</span>**

<span style="font-size: 14px; background-color: inherit;">用最<span style="color: rgb(244, 114, 7); font-size: 14px; background-color: inherit;">简明</span>的几句话说清楚希望leader配合什么事情，第一步往往是很多人做的不好的，经常的开头是这样的：</span>

“早上，我们和PM及业务方对XXX事情进行了讨论，业务方认为YYY，产品认为ZZZ，但我们认为这样会导致WWW…”

<span style="font-size: 14px; background-color: inherit;">这类**事实陈述式开头**，<span style="color: rgb(244, 114, 7); font-size: 14px; background-color: inherit;">不利于leader了解需要沟通的主题，以及需要配合的事项</span>，这样leader可能难以判断事情的紧急优先程度。</span>

&nbsp;

<span style="font-size: 14px; background-color: inherit;">如果是需要**征得leader同意**，可以这样开头：</span>

“XXX某个项目的排期，需要你确认一下，这个项目是关于YYY…”

&nbsp;

<span style="font-size: 14px; background-color: inherit;">如果是**请求leader帮助**，可以这样开头：</span>

“XXX运维工单，需要你通过一下，这个运维工单是关于YYY…”

&nbsp;

<span style="font-size: 14px; background-color: inherit;">如果是**工作汇报，或者是交流讨论**，可能需要较长的时间，不宜莽撞的临时抽空当面交流，**最好只是约一个时间**：</span>

“XXX项目的设计评审，不知道你什么时候有空，这个项目是关于YYY…”

&nbsp;

<span style="font-size: 14px; background-color: inherit;">无论如何，快速说明来意，第一时间让leader清楚需要配合什么，以及事情的紧急优先程度是什么样的，而不是一上来就陈述细节。</span>

&nbsp;

**<span style="font-size: 14px; background-color: inherit;">二，来龙去脉</span>**

<span style="font-size: 14px; background-color: inherit;">表面来意之后，需要进一步说明更多事情的细节，对来龙去脉做一个简单的描述，这个过程<span style="color: rgb(244, 114, 7); font-size: 14px; background-color: inherit;">&ldquo;逻辑性&rdquo;非常重要</span>。</span>

&nbsp;

**<span style="font-size: 14px; background-color: inherit;">事实性描述</span>**<span style="font-size: 14px; background-color: inherit;">，可以采用<span style="color: rgb(244, 114, 7); font-size: 14px; background-color: inherit;">先后时序</span>：</span>

“先xxx，然后yyy，最后zzz”

<span style="font-size: 14px; background-color: inherit;">也可以采用<span style="color: rgb(244, 114, 7); font-size: 14px; background-color: inherit;">分角色立场描述</span>：</span>

“pm认为xxx，qa觉得，我们认为zzz”

&nbsp;

**<span style="font-size: 14px; background-color: inherit;">决策性描述</span>**<span style="font-size: 14px; background-color: inherit;">，一定要<span style="color: rgb(244, 114, 7); font-size: 14px; background-color: inherit;">说明理由</span>：</span>

“建议采用xxx方案，因为xxx原因一，二，三，所以xxx方案我认为是最佳选择”

<span style="font-size: 14px; background-color: inherit;">这种&ldquo;论点，论据，强调论点&rdquo;的表达方式，逻辑性强，重点突出，强烈推荐。</span>

&nbsp;

**<span style="font-size: 14px; background-color: inherit;">三，复述确认</span>**

<span style="font-size: 14px; background-color: inherit;">对沟通目的，来龙去脉做了简要称述，最后<span style="color: rgb(244, 114, 7); font-size: 14px; background-color: inherit;">对期望leader要配合的事情做复述和确认</span>。这个过程，最重要的是**&ldquo;强烈的行动导向&rdquo;**，为leader做出的决策和行动提供强有力的信心（例如进一步计划）：</span>

“好的，如果确认使用方案XXX，后续我们将YYY”

“如果确认明天下午XXX时间有空，后续我们会YYY”

<span style="font-size: 14px; background-color: inherit;"><span style="color: rgb(244, 114, 7); font-size: 14px; background-color: inherit;">&ldquo;既然同意了XXX排期，后续我们则YYY&rdquo;</span></span>

<span style="font-size: 14px; background-color: inherit;">这个最终的确认非常重要，**所有的沟通都是为了明确最终的这个结果**，最怕误以为双方达成了一致，其实非常含糊。</span>

&nbsp;

<span style="font-size: 16px; background-color: inherit;">**【总结】**</span>

<span style="font-size: 14px; background-color: inherit;">&ldquo;快速&rdquo;&ldquo;精确&rdquo;和leader完成沟通，三个步骤：

（1）**表明期望**：简洁，明确

（2）**来龙去脉**：逻辑性强

（3）**复述确认**：行动导向</span>
