pyDash：一个基于 web 的 Linux 性能监测工具

2017-05-03 linux tools

编译自：http://www.tecmint.com/pydash-a-web-based-linux-performance-monitoring-tool/作者： Ravi Saive

<span style="text-align: left; color: rgb(153, 153, 153); text-transform: none; text-indent: 0px; letter-spacing: normal; font-family: &quot;Lantinghei SC&quot;, &quot;Helvetica Neue&quot;, &quot;Microsoft YaHei&quot;, 微软雅黑, Arial, STHeiti, &quot;WenQuanYi Micro Hei&quot;, SimSun, sans-serif; font-size: 12px; font-style: normal; font-variant: normal; font-weight: normal; word-spacing: 0px; white-space: normal; background-color: rgb(255, 255, 255); -webkit-text-stroke-width: 0px;">原创：[LCTT](https://linux.cn/lctt/)<span style="background-color: inherit;">&nbsp;</span>[https://linux.cn/article-8462-1.html](https://linux.cn/article-8462-1.html)</span><span style="text-align: left; color: rgb(153, 153, 153); text-transform: none; text-indent: 0px; letter-spacing: normal; font-family: &quot;Lantinghei SC&quot;, &quot;Helvetica Neue&quot;, &quot;Microsoft YaHei&quot;, 微软雅黑, Arial, STHeiti, &quot;WenQuanYi Micro Hei&quot;, SimSun, sans-serif; font-size: 12px; font-style: normal; font-variant: normal; font-weight: normal; word-spacing: 0px; white-space: normal; background-color: rgb(255, 255, 255); -webkit-text-stroke-width: 0px;">译者：<span style="background-color: inherit;">&nbsp;</span>[ucasFL](https://linux.cn/lctt/ucasFL)</span>

`pyDash` 是一个轻量且[基于 web 的 Linux 性能监测工具](http://www.tecmint.com/command-line-tools-to-monitor-linux-performance/)，它是用 Python 和 [Django](http://www.tecmint.com/install-and-configure-django-web-framework-in-centos-debian-ubuntu/) 加上 Chart.js 来写的。经测试，在下面这些主流 Linux 发行版上可运行：CentOS、Fedora、Ubuntu、Debian、Raspbian 以及 Pidora 。

你可以使用这个工具来监视你的 Linux 个人电脑/服务器资源，比如 CPU、内存、网络统计，包括在线用户的进程以及更多。仪表盘完全由主要的 Python 发行版本所提供的 Python 库开发，因此它的依赖关系很少，你不需要安装许多包或库来运行它。

在这篇文章中，我将展示如何安装 `pyDash` 来监测 Linux 服务器性能。
