Linux网络工具之fping

2017-03-10 linux tools

windows 类似工具PingInfoView

[http://www.nirsoft.net/utils/multiple_ping_tool.html](http://www.nirsoft.net/utils/multiple_ping_tool.html)

安装fping

yum -y install fping

查看网段里的ip情况

&nbsp;fping -Ag -s -r 1 192.168.1.1/24

Fping命令与Ping命令非常相似，使用ICMP回应请求和应答来确定系统的可达性。它提供了一些Ping没有的特殊功能.现在把常用的几个命令参数记录如下：

&nbsp;&nbsp;

fping的命令和参数详解

Usage: fping [options] [targets...]

用法：fping [选项] [ping的目标]

&nbsp;&nbsp; -a&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; show targets that are alive

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 显示可ping通的目标

&nbsp;&nbsp; -A&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; show targets by address

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 将目标以ip地址的形式显示

&nbsp;&nbsp; -b n&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; amount of ping data to send, in bytes (default 56)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ping 数据包的大小。（默认为56）

&nbsp;&nbsp; -B f&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; set exponential backoff factor to f

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 设置指数反馈因子到f 【这个不懂，求指教~】

&nbsp;&nbsp; -c n&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; count of pings to send to each target (default 1)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ping每个目标的次数 (默认为1)

&nbsp;&nbsp; -C n&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; same as -c, report results in verbose format

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 同-c, 返回的结果为冗长格式

&nbsp;&nbsp; -e&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; show elapsed time on return packets

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 显示返回数据包所费时间

&nbsp;&nbsp; -f file&nbsp;&nbsp;&nbsp; read list of targets from a file ( - means stdin) (only if no -g specified)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 从文件获取目标列表( - 表示从标准输入)(不能与 -g 同时使用)

&nbsp;&nbsp; -g&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; generate target list (only if no -f specified)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 生成目标列表(不能与 -f 同时使用)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; (specify the start and end IP in the target list, or supply a IP netmask)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; (ex. fping -g 192.168.1.0 192.168.1.255 or fping -g 192.168.1.0/24)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; (可指定目标的开始和结束IP， 或者提供ip的子网掩码)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; (例：fping -g 192.168.1.0 192.168.1.255 或 fping -g 192.168.1.0/24)

&nbsp;&nbsp; -H n&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Set the IP TTL value (Time To Live hops)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 设置ip的TTL值 (生存时间)

&nbsp;&nbsp; -i n&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; interval between sending ping packets (in millisec) (default 25)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ping包之间的间隔(单位：毫秒)(默认25)

&nbsp;&nbsp; -l&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; loop sending pings forever

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 循环发送ping

&nbsp;&nbsp; -m&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ping multiple interfaces on target host

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ping目标主机的多个网口

&nbsp;&nbsp; -n&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; show targets by name (-d is equivalent)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 将目标以主机名或域名显示(等价于 -d )

&nbsp;&nbsp; -p n&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; interval between ping packets to one target (in millisec)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 对同一个目标的ping包间隔(毫秒)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; (in looping and counting modes, default 1000)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; (在循环和统计模式中，默认为1000)

&nbsp;&nbsp; -q&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; quiet (don&#39;t show per-target/per-ping results)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 安静模式(不显示每个目标或每个ping的结果)

&nbsp;&nbsp; -Q n&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; same as -q, but show summary every n seconds

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 同-q, 但是每n秒显示信息概要

&nbsp;&nbsp; -r n&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; number of retries (default 3)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 当ping失败时，最大重试次数(默认为3次)

&nbsp;&nbsp; -s&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; print final stats

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 打印最后的统计数据

&nbsp;&nbsp; -I if&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; bind to a particular interface

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 绑定到特定的网卡

&nbsp;&nbsp; -S addr&nbsp;&nbsp;&nbsp; set source address

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 设置源ip地址

&nbsp;&nbsp; -t n&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; individual target initial timeout (in millisec) (default 500)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 单个目标的超时时间(毫秒)(默认500)

&nbsp;&nbsp; -T n&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ignored (for compatibility with fping 2.4)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 请忽略(为兼容fping 2.4)

&nbsp;&nbsp; -u&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; show targets that are unreachable

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 显示不可到达的目标

&nbsp;&nbsp; -O n&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; set the type of service (tos) flag on the ICMP packets

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 在icmp包中设置tos（服务类型）

&nbsp;&nbsp; -v&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; show version

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 显示版本号

&nbsp;&nbsp; targets&nbsp;&nbsp;&nbsp; list of targets to check (if no -f specified)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 需要ping的目标列表(不能和 -f 同时使用)

-h&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; show this page

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 显示本帮助页

&nbsp;
