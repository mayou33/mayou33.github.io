centos6搭建vnc服务器

2017-03-10 linux linux tools

安装VNC服务端

yum -y install vnc-server

配置VNC服务参数文件

# vi /etc/sysconfig/vncservers

配置如下：

# 排序数字：用户名 （自己之前添加）

VNCSERVERS=&quot;1:root 2:test1 3:test3 &quot;

VNCSERVERARGS[1]=&quot;-geometry 1024x768&quot;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;

VNCSERVERARGS[2]=&quot; -geometry 800x600&quot;

VNCSERVERARGS[3]=&quot;-geometry 1280x720&quot;

为VNC服务配置了3个用户root、test1和test2，

他们使用的端口号分别为5901、5902和5903，（默认端口从5901开始）

分辨率分别为1280x720、800x600和1280x720。（自定义分辨率）

&nbsp;&nbsp;&nbsp;&nbsp; 切换到各自的账户下使用vncpasswd来生成访问VNC服务的密码

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 先在root账户&nbsp; 分配vnc密码，然后在分配其他账户

#su &ndash; test1&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;

$vncpasswd

Password:

Verify:

这样在各自家目录下会生成一个.vnc的隐藏目录，其下面会生成一个名为passwd的存储用户密码的文件。

启动VNC服务

#&nbsp; service vncserver start&nbsp; 或者 #/etc/init.d/vncserver start

修改各用户桌面环境

启动VNC服务后在相应的用户的家目录下就会生成一个隐藏的.vnc目录和下属xstartup文件

vi /root/.vnc/xstartup

最下面有如下内容：

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; xterm -geometry 80x24+10+10 -ls -title &quot;$VNCDESKTOP Desktop&quot; &amp;

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; twm &amp;

给twm &amp;前面加上警号#。在上一行加入gnome-session&nbsp; &amp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
